Student Maths Score Predictor

An End-to-End Machine Learning Project

Project Overview

This web application utilizes advanced regression algorithms to analyze student demographic and academic data. The goal is to accurately predict a student's performance in Mathematics based on pre-existing factors.

Key Factors Analyzed

  • Gender & Race/Ethnicity
  • Parental Level of Education
  • Lunch Type (Standard vs. Free/Reduced)
  • Test Preparation Course Status
  • Reading & Writing Scores
